# Farm House Expense Tracker

A gorgeous, premium, glassmorphic web application built to track house expenses and inflows for **Eid Day 02 - Farm House event**.

---

## Features
- **Dashboard Hub**: Quick overview of Total Inflows, Total Expenses, and Net Balance (color-coded).
- **Interactive Visual Breakdown**: CSS progress indicators showing how funds are allocated across categories.
- **Expenses Ledger**: Complete transaction listing for tracking purchases. Pre-seeded with a **40,000 PKR** farmhouse expense as standard. Includes custom categories, deletion, editing, search, and filtering.
- **Inflows Ledger**: Tracks cash received, contributions, deposits, and other custom inflows.
- **Persistent Data**: Powered by browser Local Storage to keep your budget safe through page refreshes.
- **Import/Export**: Backup or share budget files as spreadsheet-friendly CSV or JSON files.
- **Fluid Layout**: Designed from the ground up to render flawlessly on desktop monitors, tablets, and smartphones.

## Setup & Running

This project uses standard vanilla HTML, CSS, and JS with zero complex compilation steps or external dependencies.

1. **Set Active Workspace**:
   Open the folder `C:\Users\muhammad.askari\.gemini\antigravity\scratch\farm-house-expense` in your IDE/editor.
2. **Launch Application**:
   Simply open the `index.html` file in any modern web browser (Chrome, Safari, Firefox, Edge).
   Alternatively, run a simple local web server of your choice:
   - For python: `python -m http.server 8000`
   - For npm: `npx live-server` or `npm install -g serve && serve`
